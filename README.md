## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository

```bash
git clone https://github.com/Bitxo92/web_portfolio.git
```

2. Install dependencies

```bash
cd web_portfolio
npm install
```

3. Start the development server

```bash
npm run dev
```

4. Build for production

```bash
npm run build
```
